# Merchant-Product-Management-System
📌 Merchant Product Management System
📝 Project Description

      The Merchant Product Management System is a console-based Java application designed to manage merchants and their products efficiently. The system allows merchants to perform CRUD operations and manage product information using a structured and scalable approach.

⚙️ Technologies Used
Core Java
Hibernate (ORM)
MySQL Database
JDBC (Internally via Hibernate)
Eclipse / IntelliJ IDEA

🚀 Features
  Add, update, and retrieve merchant details
  Add and manage products under each merchant
  CRUD operations for both Merchant and Product entities
  Merchant verification using email/password and phone/password
  Product search by ID, brand, category, and merchant ID
  Hibernate-based object-relational mapping
  Efficient data handling using SQL queries

🏗️ Project Structure
  Controller Layer – Handles user input and application flow
  DAO Layer – Contains database logic using Hibernate
  DTO Layer – Entity classes (Merchant, Product)
  Database – MySQL for persistent storage

🔍 Key Functionalities
  Merchant registration and authentication
  Product assignment to specific merchants
  Product filtering and retrieval operations
  Secure and structured database interaction using Hibernate

🎯 Learning Outcomes
  Hands-on experience in Core Java and Hibernate integration
  Understanding of ORM concepts
  Practical implementation of CRUD operations
  Database design and query optimization
